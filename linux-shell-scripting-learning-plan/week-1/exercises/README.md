# Week 1 Exercise Solutions

This directory contains solution files for Week 1 exercises.

## Exercise Files

1. **exercise_1_1_solution.sh** - Directory structure creation
2. **exercise_2_1_solution.sh** - Welcome script with user input
3. **exercise_3_1_solution.sh** - User information collector
4. **exercise_4_1_solution.sh** - Calculator script
5. **exercise_5_1_solution.sh** - File manager script
6. **exercise_6_1_solution.sh** - Batch file processor
7. **exercise_7_1_solution.sh** - System monitoring script
8. **week1_project_solution.sh** - System maintenance automation project

## Usage

Each exercise solution can be run directly:

```bash
chmod +x exercise_1_1_solution.sh
./exercise_1_1_solution.sh
```

Or viewed to understand the implementation:

```bash
cat exercise_1_1_solution.sh
```

## Learning Path

Complete exercises in order for best learning experience:
1. Exercise 1.1 (Day 1) - Basic directory operations
2. Exercise 2.1 (Day 2) - First scripts
3. Exercise 3.1 (Day 3) - Variables and input
4. Exercise 4.1 (Day 4) - Operators and arithmetic
5. Exercise 5.1 (Day 5) - Control flow
6. Exercise 6.1 (Day 6) - Loops
7. Exercise 7.1 (Day 7) - System administration
8. Week 1 Project - Comprehensive maintenance script
