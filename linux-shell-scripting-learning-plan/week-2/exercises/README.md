# Week 2 Exercise Solutions

This directory contains solution files for Week 2 exercises.

## Exercise Files

1. **exercise_1_1_solution.sh** - System monitoring function library
2. **exercise_2_1_solution.sh** - Server management with arrays
3. **exercise_3_1_solution.sh** - Log analysis script
4. **exercise_4_1_solution.sh** - Log management automation
5. **exercise_5_1_solution.sh** - Service management script
6. **exercise_6_1_solution.sh** - Scheduled maintenance system
7. **exercise_7_1_solution.sh** - Log aggregation and analysis
8. **week2_project_solution.sh** - Complete automation framework

## Learning Path

Complete exercises in order for best learning experience:
1. Exercise 1.1 (Day 1) - Functions and modular scripting
2. Exercise 2.1 (Day 2) - Arrays and data structures
3. Exercise 3.1 (Day 3) - String manipulation
4. Exercise 4.1 (Day 4) - File system operations
5. Exercise 5.1 (Day 5) - Process management
6. Exercise 6.1 (Day 6) - Cron job scheduling
7. Exercise 7.1 (Day 7) - Log aggregation
8. Week 2 Project - Comprehensive automation framework

## Running Exercises

Each exercise can be run directly:

```bash
chmod +x exercise_1_1_solution.sh
./exercise_1_1_solution.sh
```
