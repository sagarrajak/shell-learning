# Week 4: Professional System Administration - Exercises

This directory contains practice exercises for Week 4 of the Linux Shell Scripting Learning Plan.

## Exercise Files

- `exercise_1_1_solution.sh` - Log Analysis Pipeline (sed, awk, grep)
- `exercise_2_1_solution.sh` - User Management System
- `exercise_3_1_solution.sh` - Service Health Monitor
- `exercise_4_1_solution.sh` - Time Sync Configuration

## Running the Exercises

Each exercise includes a problem statement and solution. Run the solution scripts to verify your understanding.

```bash
# Make scripts executable
chmod +x exercise_*.sh

# Run specific exercise
./exercise_1_1_solution.sh
```

## Learning Objectives

By completing these exercises, you will:
- Master combining sed, awk, and grep for complex text processing
- Create user management automation scripts
- Build service monitoring systems
- Configure NTP services (chrony, ntpd, timesyncd)
